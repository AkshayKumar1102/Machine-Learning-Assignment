# Tutorial: Image Classification using Convolutional Neural Networks (CNN)
This repository contains a comprehensive tutorial on building, training, and visualizing a Convolutional Neural Network (CNN) using the CIFAR-10 dataset. 

*🎯 Project Objective
The goal of this tutorial is to demonstrate how deep learning models "see" images. We move beyond basic accuracy scores to explore **Feature Maps** and **Error Analysis**, providing a transparent look into the "Black Box" of Artificial Intelligence.

*🚀 Key Features
- **End-to-End Pipeline:** Data preprocessing, model architecture design, and evaluation.
- **Explainable AI (XAI):** Visualization of internal feature maps to show learned edge detection.
- **Accessibility Optimized:** All plots use high-contrast colors, distinct line styles, and markers for color-blind accessibility.
- **Error Analysis:** Visual identification of misclassified images to understand model limitations.

*🛠️ How to Run
1. Open the provided `.ipynb` file in [Google Colab](https://colab.research.google.com/).
2. Ensure the Runtime is set to **GPU** (Runtime > Change runtime type > T4 GPU).
3. Run all cells sequentially.
